HTML - Lenguaje de programacao de hipertexto

Estrutura Basica

Documentos HTML sao divididos em algumas partes basicas. Antes de iniciar os elementos, devemos utilizar uma declaracao <!DOCTYPE html≷ na primeira linha.

O elementos raiz "html" e de onde partem todos os outros elementos ele e composto s partes "head" e "body".

Tags e elementos

Na marcacao HTML, cada elemento e representado por uma tag de abertura, o conteudo e uma tag de fechamento. Tags de abertura sao escritas no formato <.; nome-da-tag> tags de fechamento no formato </nome-da-tag>

Elementos de texto

Alguns elemtnso de texto que podemos utilizar:

  • paragrafo - tag <p>
  • titulo - tags <h1> a <h6>
  • links - tag <a>

Criando uma nova pagina

Para criar uma nova pagina devemos:

  1. Criar um novo arquivo con extensao .html
  2. Inserir a declaracao octype na primeira linha
  3. riar os elementos html, head e body
  4. Inserir o conteudo
